# Mathematics for Machine Learning

This repository contains **practical implementations** of the mathematical concepts from the book [*Mathematics for Machine Learning*](https://mml-book.github.io/) by Marc Peter Deisenroth, A. Aldo Faisal, and Cheng Soon Ong.

## Why This Repository?

Machine learning is built on mathematics. Understanding the math behind the algorithms helps you:
- Build better machine learning models
- Debug and improve your code
- Make informed decisions about which algorithm to use
- Understand what's happening under the hood

## What You'll Find Here

This repository breaks down mathematical concepts from the book into **simple, working examples**. Each implementation is written to be:
- **Easy to understand** — no unnecessary jargon
- **Practical** — code you can run and learn from
- **Accessible** — for anyone interested in machine learning

## Getting Started

Check out the code examples and run them yourself. Each piece of code is designed to teach, not just compute.
